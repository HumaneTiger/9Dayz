Game Editor

1. Select a square in the game world

2. Select the object you want to place

3. Place it in the selected square

The new object behaves like any pre-placed object. If you place it in direct range of the character, its Card will spawn instantly.

If you place it anywhere else, you have to move your character close to that place to make its Card spawn.

Please note: Only objects placed next to or on a street will be seen by the character and spawn.
